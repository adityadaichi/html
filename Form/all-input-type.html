TEXT INPUT type="text" creates a single-line text box for user input. placeholder="Enter your name" shows hint text until user types. maxlength="20" limits the input length.

PASSWORD INPUT type="password" hides typed characters with dots or asterisks.

EMAIL INPUT type="email" requires a valid email format (e.g., user@example.com).

NUMBER INPUT type="number" allows only numeric values. min, max, and step control the range.

TEL INPUT type="tel" is for telephone numbers (no validation by default).

URL INPUT type="url" requires a valid URL format (e.g., https://example.com).

SEARCH INPUT type="search" is a single-line text box with optimized styling for search.

DATE INPUT type="date" provides a date picker.

TIME INPUT type="time" provides a time picker.

DATETIME-LOCAL INPUT type="datetime-local" lets users pick date and time (no timezone).

MONTH INPUT type="month" lets users pick month and year.

WEEK INPUT type="week" lets users pick a week of a year

COLOR INPUT type="color" shows a color picker.

RANGE INPUT type="range" is a slider for numeric values. min, max, and step control movement.

CHECKBOX INPUT type="checkbox" allows multiple selections.

RADIO INPUT type="radio" allows single selection among options with same name.

FILE INPUT type="file" lets user upload files. accept restricts file types. multiple allows multiple uploads.

HIDDEN INPUT type="hidden" stores data not visible to the user.

SUBMIT BUTTON type="submit" sends form data to server.

RESET BUTTON ype="reset" clears all form inputs.

BUTTONtype="button" is a general button (no default behavior).
